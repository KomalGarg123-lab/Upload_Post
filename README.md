Upload Post Application (MERN)

This project is a simple social media style web application where users can upload posts with an image and caption. The uploaded posts are stored in the database and displayed in a feed page.

The application is built using React for the frontend and Node.js with Express for the backend. Images are uploaded using Multer and stored using a storage service.

Features

Create a post with an image and caption

Upload images from the frontend

Store post data in MongoDB

Fetch and display all posts in a feed

Automatic navigation to the feed after creating a post
